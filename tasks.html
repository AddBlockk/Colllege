Задание

  1. Создайте файл my.html со своими данными: имя, фамилия отчество, группа, специальность. Используйте форматирование текста (выравнивание, кегль).
  2. В файл my.html добавьте три горизонтальных линии после фамилии, имени, отчества. После этого введите столбиком, изучаемые предметы этой сессии.
  3. Преобразуйте список предметов в нумерованный список. Создайте маркированный список своих интересов (хобби), сопроводив его заголовком.
  4. Создайте другой файл, назовите его zadan.html. Откройте свой файл и создайте в нем ссылку с именем «задания по практической работе». В файл zadan.html укажите все задания, которые перечислены в данной практической работе
  5. Найдите на компьютере графические файлы и включите их в состав своего файла my.html.
  6. Используя разные цветовые гаммы определите цвет фона и цвет каждого шрифта, используемого в документе.
  7. Создайте в своем документе таблицу расписания занятий на текущую неделю.
